# Basic Syntax

### Heading

* \#H1
* \#\#H2
* \#\#\#H3
* ...
* \#\#\#\#\#\#H6

### Bold Text

* \*\*_bold_ _text goes here_\*\*

### Italic Text

* \*_italicized text goes here\*_

### Ordered List

* `1.`Item 1
* `2.`Item 2
* `3.`Item 3

### Unordered List

* `-` Item 1
* * `-` `-` Item 2
    * `-` `-` `-` Item 3

### Code

```text
´code goes here´
```



### Horizontal rule

 `-` `-` `-` 



### Link

\[title\]\([www.google.com](https://www.example.com)\)



### Image

!\[alt text\] \(image.png\)
